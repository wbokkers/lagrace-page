Psaume 3 Dawuda xa Ala maxandi 


Alatala, n yaxuie bara wuya
N gerefae bara gbo
Mixi wuyaxi a falama
Ala mu nɔma n nakiside
Kɔnɔ I tan Alatala, I findixi n kantama nan na
N ma binyɛ nun ma xunnakeli fatanxi I tan nan na

N n xui ratema Ala ma
A n yaabima keli a xa geya sɛniyɛnxi fari

N nan n sama, n xa xi
N man xunuma,
Barima Alatala n dɛmɛnma nɛ
N mu gaaxuma ɲama ya ra
Naxee mu wama n ma fe xɔn ma
Alatala yandi, n nakisi, n Marigi Ala
I n yaxuie dɛ ragarinma nɛ,
I mixi ɲaaxi ɲinyie magirama nɛ.
Marakisima nan na Alatala ra.
