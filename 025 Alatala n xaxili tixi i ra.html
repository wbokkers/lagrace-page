Psaume 25B Alatala n xaxili tixi



Alatala n xaxili tixi i ra.

N Marigi Alatala, n bara n yɛtɛ taxu i ra abadan.

Alako n naxa yaagi de.

Alako n yaxuie naxa n mayele.

Naxan yo la i ra, na kanyi mu yaagima.

Kɔnɔ yanfante tan go.

E ya rabima nɛ, abadan.



Alatala n tixi i xa kira xɔn ma.

I xa n xaran i xa kira ra.

Alatala I tan nan n nakisima.

N xaxili tixi i ra lɔxɔɛ birin.
