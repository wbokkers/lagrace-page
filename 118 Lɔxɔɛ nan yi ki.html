Psaume 118 Lɔxɔɛ nan yi ki


	
1. Lɔxɔɛ nan yi ki, 	  (2x) 
Ala naxan daaxi, 	  (2x) 
Won xa ɲɛlɛxin, 	  (2x) 
won xa sɛɛwa a kui, (2x) 
Lɔxɔɛ nan yi ki Ala naxan daaxi. 
Won xa ɲɛlɛxin, won xa sɛɛwa a kui. 
Lɔxɔɛ nan yi ki, lɔxɔɛ nan yi ki Ala naxan daaxi. 

2. Yandi Ala,                (2x) 
muxu xun nakeli,     (2x)
Yandi Ala, yandi Ala, muxu rakisi, muxu rakisi. 
Yandi Ala muxu xun nakeli. 
Yandi Ala, muxu rakisi. 
Yandi Ala, Yandi Ala, muxu xunnakeli. 


3. Marigi, Marigi nan na Ala ra, nan na Ala ra. 
Naxan xa nɔrɛ,          (2x) 
bara yanba won ma,  (2x) 
Marigi nan na Ala ra. 
Naxan xa nɔrɛ bara yanba won ma. 
Marigi, Marigi nan na Ala ra. 

4. N Marigi Ala,      (2x)
n i matɔxɔma nɛ, (2x)
N Marigi Ala,      (2x) 
n i tantuma nɛ,     (2x)
N Marigi Ala n i matɔxɔma nɛ. 
N Marigi Ala n i tantuma nɛ. 
N Marigi Ala, n Marigi Ala, n i matɔxɔma nɛ. 


5. Won xa Ala tantu! 
Won xa Ala tantu, barima a fan, barima a fan. 
A xa xanunteya,   (2x) 
mu ɲɔnma nɛ,       (2x) 
Won xa Ala tantu, barima a fan. 
A xa xanunteya mu ɲɔnma nɛ. 
Won xa Ala tantu! Won xa Ala tantu barima a fan.
